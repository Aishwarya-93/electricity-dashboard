# ⚡ Electricity Consumption Dashboard

An interactive data analytics dashboard built using **Streamlit** to analyze electricity consumption patterns, detect anomalies, and visualize insights including a geospatial map of India.

---

## 🚀 Features

- 📂 Upload your own CSV dataset  
- 📊 Electricity consumption trend visualization  
- 🔥 Peak usage analysis  
- ⚠️ Anomaly detection using Z-score  
- 📈 Prediction using Machine Learning  
- 🗺️ India map visualization (state-wise consumption)  
- 📥 Download processed data & anomaly reports  

---

## 🛠 Tech Stack

- **Python**
- **Streamlit**
- **Pandas**
- **Plotly**
- **Scikit-learn**

---

## 📊 Key Insights

- Identifies peak electricity consumption periods  
- Detects unusual spikes or drops (anomalies)  
- Provides state-wise comparison using geospatial visualization  
- Helps in understanding long-term consumption trends
